# 眷顾大纲

# 江城

这个无名的小城，这个初中的孩子，这条注定奔流到海的江。

他的房间里放了喜欢的手办，但最新买来的手办似乎是一个神像。

神像扭曲了他的房间，手办从此有了生命，并且他的日常生活也开始扭曲，家人，朋友，同学，现实对他来说变得像是达利的《记忆的永恒》。

看起来善良的手办告诉他，这个世界的时间正在受到侵蚀，作为比喻来说，侵蚀就像癌细胞扩散一样，所有人的时间都被隔离开了，这是时间的免疫系统攻击自身导致的现象，物质和精神的世界被模糊。

他眼睁睁看着不好的幻想成真，自己和身边的人遭遇或恐怖或光怪陆离的事件，渐渐失去世界的真实，让自己暗恋的女生遭遇了不幸。而自己的父母，也在某日的恐怖事件（重复/复制）中失踪。

到最后，他的房间反而成为最稳定的锚点。

每个人虽然身处时间之中，但对于时间的感知却是不同的，人类每个个体对时间存在着独特的观察，那个孩子恰好有着观察“时间之浪”的天赋。

最终，他带着手办赠予的一把刀，手握那个最初的神像，坐上了行驶在江滩边的一辆大巴，前往未知的时空。

这是一场关于时间的”板块移动“，而孩子是这场灾难的难民。

# 列车

日复一日的宿醉。

他怀中揣着神像，手里紧握着刀，在列车中厮杀。

这里是否是某场游戏，人们的身上，漂浮着符文，而符文倒数着这里的生命。

他是时间的难民，他沉沦在不属于自己的时空。

这是生命的倒数吗，还是沙漏本身的重置？
